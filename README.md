# TriggerLab Reaction Arena

TanStack React + React Three Fiber reaction tester for standard Gamepad API controllers.

- Real controller model loaded from `public/models/dualshock.glb`
- Random face-button prompts with 0.8-2.6s anticipation delay
- 2.5s timeout, wrong-button misses, reaction grading, streak remarks
- Last 8 game summaries with top-3 medals
- Animation vocabulary captured in `skills/animation-vocabulary.md`
